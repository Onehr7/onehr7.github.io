Python虚拟环境配置
2020-11-16|python

  1. 安装 virtualenv

    pip install virtualenv

  2. 切换到项目目录下，创建虚拟环境

    Windows: virtualenv -p python venv
linux: sudo virtualenv -p python3 venv

  3. 激活虚拟环境

    # windows
cd venv/Scripts
activate

# linux
source venv/bin/activate

  4. 生成requirement文件

    pip freeze > requirements.txt

  5. 安装requirements.txt所需的模块

    pip install -r requirements.txt

  6. 退出虚拟环境

    deactivate
