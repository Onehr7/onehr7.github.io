2011. 执行操作后的变量值

思路

​ 水题，常规循环+判断

解答
class Solution:
    def finalValueAfterOperations(self, operations: List[str]) -> int:
        res = 0
        for operation in operations:
            if operation == '++X' or operation == 'X++':
                res += 1
            elif operation == '--X' or operation == 'X--':
                res -= 1
        return res

2012. 数组美丽值求和

思路

​ 分两次前后循环，得出数组的正向和反向的最大最小值列表，再判断得出最终的美丽值。

解答
class Solution:
    def sumOfBeauties(self, nums: List[int]) -> int:
        nums_max = [nums[0]] * len(nums)
        nums_min = [nums[-1]] * len(nums)

        for i in range(1, len(nums)):
            nums_max[i] = max(nums[i], nums_max[i - 1])

        for i in range(len(nums) - 2, -1, -1):
            nums_min[i] = min(nums[i], nums_min[i + 1])

        res = 0
        for i in range(1, len(nums) - 1):
            if nums_max[i - 1] < nums[i] < nums_min[i + 1]:
                res += 2
            elif nums[i-1] < nums[i] <  nums[i+1]:
                res += 1
        return res

2013. 检测正方形

思路

​ 使用Hash表存储每次添加的点， 每次查询时， 判断能构成正方形的三个点是否存在哈希表中，判断公式如下：x,y(查询的点)， (x1,y1), (x, y1), (x1,y)存在哈希表中，且abs(x1 - x) = abs(y1 - y)

解答
from collections import Counter
class DetectSquares:

    def __init__(self):
        self.square = Counter()


    def add(self, point: List[int]) -> None:
        x, y = point
        self.square[(x, y)] += 1

    def count(self, point: List[int]) -> int:
        x1, y1 = point
        res = 0
        for x, y in self.square:
            if abs(x1-x) == abs(y1-y) and x != x1:
                 if (x1, y) in self.square and (x, y1) in self.square:
                     res += self.square[(x, y)] * self.square[(x1, y)] * self.square[(x, y1)]
        return res
