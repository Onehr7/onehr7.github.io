二叉搜索树
2019-04-24|数据结构

二叉搜索树（BST）

特性

一颗二叉树可以为空，当不为空时：

  • 非空左子树的所有键值小于其根节点的键值

  • 非空右子树的所有键值大于其根节点的键值

  • 左右子树都是二叉搜索树


功能实现

查找
  • 树为空时，返回null
  • 树非空，与根结点关键字比较
  • 最大最小查找，找最右最左的元素
插入
  • 与根结点比较，直到根结点子树为空
删除
  • 叶结点：直接删除
  • 只有一个孩子结点：直接删除，将被删除结点的子节点指向被删除结点的父结点
  • 有两个孩子结点：用右子树最小元素或左子树最大元素代替被删除结点

可视化演示地址：https://visualgo.net/zh/bst


相应函数

查找- -递归实现FindByRec
查找- -非递归实现FindByFuc

    private Node root;          //二叉搜索树的根结点

    private class Node&#123;

        int val;
        private Node left;
        private Node right;

        public Node(int val, Node left, Node right)&#123;
            this.val = val;
            this.left = left;
            this.right = right;

    /**
     * 查找
     * --递归实现
     */
    public Node FindByRec(int value)&#123;
        return FindByRec(root,value);
    &#125;

    private Node FindByRec(Node node, int value)&#123;

        if(root == null)&#123;
            return null;
        &#125;

        if(root.val > value)&#123;
            return FindByRec(root.right, value);
        &#125;

        else if(root.val < value)&#123;
            return FindByRec(root.left, value);
        &#125;

        else&#123;
            return node;
        &#125;
    &#125;

    *查找
    * 非递归
    */

    public Node FindByFuc(int value)&#123;
        return FindByFuc(root, value);
    &#125;

    private Node FindByFuc(Node node, int value)&#123;
        while (node != null)&#123;
            if(node.val > value)&#123;
                  node = node.left;
            &#125;

            else if(node.val < value)&#123;
                node = node.right;
            &#125;

            else&#123;
                return node;
            &#125;
        &#125;
        return node;
    &#125;
