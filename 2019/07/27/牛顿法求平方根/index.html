牛顿迭代法求平方根
2019-07-27|数据结构

x 的平方根

要点

牛顿迭代法
  • 求x的平方根即求f(x)=x^2-a等于0的解，
  • 通过点x的直线于曲线，在无限趋近于x的时候，直线与曲线近似相等
  • 点x处切线交x轴于x0
  • 作x0的垂线交曲线与(x0,f(x0))
  • 通过反复迭代求得x的平方根近似值

要求

实现 int sqrt(int x) 函数。

计算并返回 x 的平方根，其中 x 是非负整数。

由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。


示例

输入: 4
输出: 2


输入: 8
输出: 2
说明: 8 的平方根是 2.82842..., 
由于返回类型是整数，小数部分将被舍去。

代码


class Solution &#123;
    public int mySqrt(int x) &#123;
       long a = x;
       while (a * a > x)&#123;
           a = (a + x / a) / 2 ;
       &#125;
    return (int)a;
