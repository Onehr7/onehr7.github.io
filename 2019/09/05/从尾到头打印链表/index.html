从尾到头打印链表
2019-09-05|剑指offer

题目描述

输入一个链表，按链表从尾到头的顺序返回一个ArrayList


#### 问题分析
  • 先将链表数据全部压入栈中
  • 通过运用栈后入先出的性质
  • 即可将数据反转

#### 知识点 * ArrayList list = new ArrayList(); * stack.push() || pop() * list.add(value);

代码实现

/**
*    public class ListNode &#123;
*        int val;
*        ListNode next = null;
*
*        ListNode(int val) &#123;
*            this.val = val;
*        &#125;
*    &#125;
*
*/
import java.util.ArrayList;
import java.util.Stack;
public class Solution &#123;
    public ArrayList<Integer> printListFromTailToHead(ListNode listNode) &#123;
        Stack<Integer> stack = new Stack<>();
        ArrayList<Integer> list = new ArrayList<Integer>();

        //将链表中的数据压入栈中
        while(listNode!=null)&#123;
            stack.push(listNode.val);
            listNode = listNode.next;
        &#125;

        //将栈中数据返回给ArrayList
        while(!stack.empty())&#123;
            list.add(stack.pop());
        &#125;
        return list;
    &#125;
